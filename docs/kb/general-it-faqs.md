# General IT FAQs

## Wi-Fi

Connect to the `Company-Corp` SSID with your SSO credentials. The `Company-Guest` SSID is for visitors only and uses a daily-rotating password posted at the front desk. Macs may need to "forget" the network and re-join after a password rotation.

## Escalation tiers

IT support has three tiers. Tier 1 (this assistant + the help desk) handles routine issues like access requests, lockouts, and ticket status. Tier 2 (named IT engineers) handles complex incidents and config changes. Tier 3 (Security and Infra leads) handles outages, breaches, and policy escalations. The assistant escalates to Tier 2 by default; Security cases go straight to Tier 3.

## Contacting IT

The fastest way to reach IT is this assistant or the `#it-support` Slack channel. For urgent in-person help, the IT desk is on the 3rd floor and staffed weekdays 9-5. For after-hours P1 incidents, page the on-call engineer via PagerDuty (link in the IT Slack channel topic).
